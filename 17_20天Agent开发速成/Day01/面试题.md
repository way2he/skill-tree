## ✅ 答题注意事项

1. **先说结论，再说细节**：不要上来就长篇大论，先一句话总结，再展开
2. **结合 Agent 开发场景**：异步在 Agent 开发中主要用在什么地方？批量调用大模型API！
3. **有量化数据最好**：比如"同步100次要100秒，异步优化后只要5-10秒，提升10-20倍"
4. **代码题注意风格**：变量名清晰，加注释，边界条件考虑周全
