## 一、概念题（共40分，每题8分）

### 第1题：协程 vs 线程（8分）
**问题**：请说明协程（Coroutine）和线程（Thread）的主要区别，至少列举3个维度。

<details>
<summary>💡 参考答案与打分细则</summary>

| 评分点 | 分值 | 说明 |
|--------|------|------|
| 调度方式 | 2分 | 协程是用户态调度，线程是内核态调度 |
| 切换开销 | 2分 | 协程切换开销极小（函数级），线程切换开销大（上下文切换） |
| 并发数量 | 2分 | 协程可轻松开10000+，线程几百个就到瓶颈 |
| 数据安全 | 2分 | 协程在单线程内不需要锁，线程需要锁机制 |

**评分标准**：答对1个维度得2分，3个及以上得8分，言之有理即可。
</details>

## 二、代码题（共40分，每题20分）

### 第6题：异步函数纠错（20分）
**问题**：下面代码有3处问题，请找出来并修正。

```python
# 请找出以下代码的问题并修正
import asyncio
import aiohttp

async def fetch(url):
    # 问题1在哪里？
    async with aiohttp.get(url) as response:
        return await response.text()

async def main():
    urls = ["url1", "url2", "url3"]
    for url in urls:
        # 问题2在哪里？
        await fetch(url)
    
    # 问题3在哪里？
    session = aiohttp.ClientSession()
    async with session.get("url4") as resp:
        print(await resp.text())

asyncio.run(main())
```

<details>
<summary>💡 参考答案与打分细则</summary>

**3处问题（各6分，共18分，代码风格2分）**：

| 问题 | 分值 | 说明 |
|------|------|------|
| 问题1 | 6分 | `aiohttp.get()` 错误，应该是 `session.get()`，必须通过 ClientSession 调用 |
| 问题2 | 6分 | for 循环里 await 是串行执行，应该用 `asyncio.gather(*tasks)` 并发 |
| 问题3 | 6分 | ClientSession 没有复用，应该在 main 开头创建，整个程序共用一个 |
| 代码风格 | 2分 | 修正后代码结构清晰，符合最佳实践 |

**修正后的正确代码**：
```python
async def fetch(session, url):
    async with session.get(url) as response:
        return await response.text()

async def main():
    urls = ["url1", "url2", "url3"]
    async with aiohttp.ClientSession() as session:
        tasks = [fetch(session, url) for url in urls]
        await asyncio.gather(*tasks)
```
</details>

### 第7题：实现带超时的并发请求（20分）
**问题**：请实现一个函数，要求：
1. 并发请求 10 个 URL
2. 最大并发数限制为 3
3. 每个请求超时时间为 5 秒
4. 返回每个 URL 的结果（成功返回内容，失败返回错误信息）

<details>
<summary>💡 参考答案与打分细则</summary>

| 评分点 | 分值 | 说明 |
|--------|------|------|
| 信号量控制并发 | 7分 | 正确使用 `asyncio.Semaphore(3)` |
| 超时控制 | 5分 | 正确使用 `ClientTimeout(total=5)` |
| gather 并发执行 | 5分 | 正确创建任务列表并并发执行 |
| 错误处理 | 3分 | try-except 捕获异常并返回错误信息 |

**参考实现核心代码**：
```python
async def fetch_with_semaphore(session, url, semaphore):
    async with semaphore:
        try:
            async with session.get(url) as response:
                return await response.text()
        except Exception as e:
            return f"Error: {str(e)}"

async def main():
    semaphore = asyncio.Semaphore(3)
    timeout = ClientTimeout(total=5)
    async with aiohttp.ClientSession(timeout=timeout) as session:
        tasks = [fetch_with_semaphore(session, url, semaphore) for url in urls]
        return await asyncio.gather(*tasks)
```

**评分标准**：核心实现正确得18-20分，缺一个功能扣对应分数。
</details>
