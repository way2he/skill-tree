# Day12 面试题（6 道）📝

&gt; ⏰ **建议时间**：1 小时  
&gt; 🎯 **目标**：自己先答，再看标准答案

---

## 基础题（3 道）

### 第 1 题（必问）
**问题**：请介绍多 Agent 协作有哪些常见的架构模式？分别有什么优缺点？

**答题要求**：
- 至少说出 3 种模式
- 对比它们的优缺点
- 给出适用场景

---

### 第 2 题（常问）
**问题**：请介绍 CrewAI 的核心概念（Agent、Task、Crew、Process），以及它们之间的关系？

**答题要求**：
- 解释每个概念的作用
- 说明它们如何配合工作
- 给出简单的代码示例（伪代码即可）

---

### 第 3 题（常问）
**问题**：如何设计一个好的 Agent 角色？需要考虑哪些要素？请举例说明。

**答题要求**：
- 列出关键要素
- 说明每个要素的作用
- 给出一个完整的 Agent 定义示例

---

## 进阶题（3 道）

### 第 4 题
**问题**：单 Agent 和多 Agent 如何选择？分别适用于什么场景？请举例说明。

**答题要求**：
- 给出选择的判断标准
- 对比单 Agent 和多 Agent 的优缺点
- 举 2-3 个例子说明

---

### 第 5 题
**问题**：在多 Agent 协作中，如何设计任务之间的依赖关系？有哪些常见的编排策略？

**答题要求**：
- 说明如何定义任务依赖
- 介绍几种编排策略
- 给出一个有依赖关系的任务设计示例

---

### 第 6 题（实战题）
**问题**：假设你要搭建一个"内容创作团队"的多 Agent 系统，请设计：
1. 需要哪些 Agent 角色？
2. 每个 Agent 的目标和背景故事？
3. 任务如何分配和编排？
4. 整体架构图？

**答题要求**：
- 详细说明设计思路
- 给出具体的 Agent 定义
- 画出任务流程图

---
