# Day13 面试题（6 道）📝

## 基础题（3 道）

### 第 1 题
什么是协调者模式？它有什么优缺点？适用什么场景？

### 第 2 题
多个 Agent 之间产生意见冲突怎么办？请介绍 3 种以上的解决策略。

### 第 3 题
如何让多个 Agent 共享信息？请介绍几种实现方式。

---

## 进阶题（3 道）

### 第 4 题
请设计一个"评审委员会"系统：
- 多个专家 Agent 评审一个方案
- 如何收集各方意见？
- 如何达成最终决策？

### 第 5 题
在多 Agent 系统中，如何避免"群体思维"？如何保证意见多样性？

### 第 6 题
Agent 之间如何进行有效的协商？请设计一个协商流程，让两个有冲突的 Agent 达成一致。
