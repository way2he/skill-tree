# 共享工作空间 📋

## 为什么需要共享工作空间？

- Agent 之间需要共享信息
- 需要记录进度和状态
- 需要避免重复工作

## 共享内容

| 类型 | 说明 |
|------|------|
| **共享记忆** | 对话历史、之前的决策 |
| **共享状态** | 进度、阶段、当前状态 |
| **共享资源** | 文档、数据、工具 |
| **共享成果** | 已完成的工作、中间产出 |

## 实现方式

### 1. 用全局字典

```python
shared_workspace = {
    "progress": "初期阶段",
    "documents": [],
    "decisions": [],
    "current_task": None
}

# Agent 1 更新
shared_workspace["progress"] = "进行中"
shared_workspace["documents"].append("document1")

# Agent 2 读取
progress = shared_workspace["progress"]
```

### 2. 用 CrewAI 的 memory

```python
crew = Crew(
    agents=[...],
    tasks=[...],
    memory=True,  # 开启记忆
    verbose=True
)
```

### 3. 用外部存储

- Redis：共享缓存
- 数据库：持久化存储
- 文件系统：共享文件
