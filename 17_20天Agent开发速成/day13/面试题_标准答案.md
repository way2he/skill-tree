# Day13 面试题标准答案 ✅

## 第 1 题：协调者模式

**答案要点**：

| 维度 | 说明 |
|------|------|
| **定义** | 有一个中心协调者 Agent，负责分配任务、收集反馈、做出决策 |
| **优点** | 有统一指挥，避免混乱；责任清晰；容易做最终决策 |
| **缺点** | 协调者可能成为瓶颈；如果协调者能力不足，会影响整体 |
| **适用** | 复杂项目、需要统一决策、有明确上下级的场景 |

---

## 第 2 题：冲突解决策略

**答案要点**：

1. **投票制** - 多个 Agent 投票，少数服从多数
2. **层级制** - 高级别的 Agent 做最终决定
3. **协商制** - Agent 之间对话协商
4. **仲裁制** - 请第三方 Agent 来仲裁

---

## 第 3 题：信息共享

**答案要点**：

1. **共享字典/对象** - 简单的全局变量
2. **CrewAI memory** - 框架内置的共享记忆
3. **外部存储** - Redis、数据库、文件系统

---

## 第 4 题：评审委员会设计

**答案要点**：

```
1. 定义多个专家 Agent（技术、产品、商业、设计）
2. 每个专家从自己的角度评审
3. 收集所有专家的意见
4. 用投票制或层级制做最终决策
5. 输出最终的决策和理由
```

---

## 第 5 题：避免群体思维

**答案要点**：

1. **指定唱反调的人** - 专门有一个 Agent 负责反对
2. **鼓励独立思考** - 让 Agent 先独立思考，再汇总
3. **引入外部视角** - 请外部 Agent 提供不同意见
4. **奖励多样性** - 明确鼓励不同意见

---

## 第 6 题：协商流程设计

**答案要点**：

```
1. 各自陈述立场和理由
2. 互相提问和澄清
3. 寻找共同点
4. 提出折衷方案
5. 评估折衷方案
6. 达成一致（或找仲裁）
```
