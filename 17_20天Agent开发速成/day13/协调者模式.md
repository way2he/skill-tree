# 协调者模式 🎯

## 什么是协调者模式？

有一个中心协调者 Agent，负责：
- 分配任务
- 收集反馈
- 做出决策
- 协调冲突

```mermaid
graph TB
    A[协调者 Agent] --&gt; B[Agent 1]
    A --&gt; C[Agent 2]
    A --&gt; D[Agent 3]
    B --&gt; A
    C --&gt; A
    D --&gt; A
```

## 协调者的职责

| 职责 | 说明 |
|------|------|
| **任务分配** | 把任务分给合适的 Agent |
| **进度追踪** | 了解每个 Agent 的进度 |
| **冲突协调** | 当 Agent 意见不一致时协调 |
| **最终决策** | 在多个方案中做出选择 |
| **信息汇总** | 把多个 Agent 的输出整合 |

## 代码示例

```python
coordinator = Agent(
    role="项目协调人",
    goal="协调团队，做出最终决策",
    backstory="你是一位经验丰富的协调者，善于倾听不同意见，做出明智决策。"
)

# 在 CrewAI 中用 hierarchical 模式
crew = Crew(
    agents=[coordinator, agent1, agent2, agent3],
    tasks=[...],
    process=Process.hierarchical,
    manager_llm=llm
)
```
