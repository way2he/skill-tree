# Day18 今日串讲 🎉

## 内容回顾

- FastAPI：快速构建API，自动文档
- SSE：服务器推送，流式输出
- Docker：容器化部署
- Gradio/Streamlit：快速构建UI

## 面试要点

1. FastAPI的优点？
   - 自动API文档
   - 类型提示
   - 高性能
   - 异步支持

2. Docker的优势？
   - 环境一致性
   - 轻量
   - 易部署

3. Gradio vs Streamlit？
   - Gradio：适合快速演示、ML demo
   - Streamlit：适合数据应用、交互式分析
