## 一、常见错误类型

### 1. 语法错误（SyntaxError）

**示例：**
```python
# 错误：缺少冒号
if True
    print("Hello")

# 错误：括号不匹配
print("Hello"

# 错误：缩进错误
def foo():
print("Hello")
```

**特征：**
- 代码还没开始运行就报错
- 错误信息包含 `SyntaxError`
- 通常有行号指示错误位置

---

### 2. 名称错误（NameError）

**示例：**
```python
# 错误：变量未定义
print(unknown_variable)

# 错误：函数未定义
unknown_function()
```

**特征：**
- 错误信息：`name 'xxx' is not defined`
- 可能是拼写错误、作用域问题、忘记导入

---

### 3. 类型错误（TypeError）

**示例：**
```python
# 错误：类型不匹配
1 + "2"

# 错误：参数数量不对
len()

# 错误：不可调用
5()
```

**特征：**
- 错误信息：`unsupported operand type(s)` 或 `must be str, not int`
- 类型不匹配或操作错误

---

### 4. 索引错误（IndexError）

**示例：**
```python
# 错误：索引越界
lst = [1, 2, 3]
print(lst[10])

# 错误：空列表索引
lst = []
print(lst[0])
```

**特征：**
- 错误信息：`list index out of range`
- 访问了不存在的索引

---

### 5. 键错误（KeyError）

**示例：**
```python
# 错误：键不存在
d = {"a": 1}
print(d["b"])
```

**特征：**
- 错误信息：`KeyError: 'xxx'`
- 访问了字典中不存在的键

---

### 6. 属性错误（AttributeError）

**示例：**
```python
# 错误：属性不存在
s = "hello"
s.unknown_method()

# 错误：NoneType没有属性
x = None
x.foo()
```

**特征：**
- 错误信息：`'xxx' object has no attribute 'yyy'`
- 访问了不存在的属性或方法

---

### 7. 值错误（ValueError）

**示例：**
```python
# 错误：无效的值
int("abc")

# 错误：解包数量不匹配
a, b = [1, 2, 3]
```

**特征：**
- 错误信息：`invalid literal for int()` 或 `too many values to unpack`
- 类型正确但值无效

## 二、报错分析的完整流程

### 流程图

```
捕获错误信息
  ↓
解析错误类型和位置
  ↓
理解错误原因
  ↓
定位问题代码
  ↓
生成修复建议
```

### 每个步骤的说明

| 步骤 | 说明 |
|------|------|
| **捕获错误信息** | 获取完整的traceback |
| **解析错误类型和位置** | 识别错误类型、行号、文件 |
| **理解错误原因** | 理解为什么会出现这个错误 |
| **定位问题代码** | 找到出错的具体代码位置 |
| **生成修复建议** | 给出具体的修复方案 |

## 三、错误分析器实现

### 1. 基础错误分析器

```python
import traceback
import re

class ErrorAnalyzer:
    def __init__(self):
        pass
        
    def analyze(self, code, error_traceback):
        # 解析traceback
        tb_lines = error_traceback.strip().split("\n")
        
        # 提取错误类型和消息
        error_line = tb_lines[-1]
        error_type, error_msg = self._parse_error_line(error_line)
        
        # 提取错误位置
        error_location = self._extract_error_location(tb_lines)
        
        # 分析原因
        analysis = self._analyze_cause(error_type, error_msg, code, error_location)
        
        return {
            "error_type": error_type,
            "error_message": error_msg,
            "error_location": error_location,
            "analysis": analysis
        }
        
    def _parse_error_line(self, line):
        match = re.match(r"(\w+):\s*(.+)", line)
        if match:
            return match.group(1), match.group(2)
        return "UnknownError", line
        
    def _extract_error_location(self, tb_lines):
        for line in reversed(tb_lines):
            match = re.search(r'File "(.+?)", line (\d+)', line)
            if match:
                return {
                    "file": match.group(1),
                    "line": int(match.group(2))
                }
        return None
        
    def _analyze_cause(self, error_type, error_msg, code, error_location):
        if error_type == "NameError":
            return self._analyze_name_error(error_msg, code, error_location)
        elif error_type == "TypeError":
            return self._analyze_type_error(error_msg, code, error_location)
        elif error_type == "IndexError":
            return self._analyze_index_error(error_msg, code, error_location)
        elif error_type == "KeyError":
            return self._analyze_key_error(error_msg, code, error_location)
        elif error_type == "AttributeError":
            return self._analyze_attribute_error(error_msg, code, error_location)
        elif error_type == "SyntaxError":
            return self._analyze_syntax_error(error_msg, code, error_location)
        else:
            return self._analyze_generic_error(error_type, error_msg, code, error_location)
            
    def _analyze_name_error(self, error_msg, code, error_location):
        match = re.search(r"name '(.+?)' is not defined", error_msg)
        if match:
            var_name = match.group(1)
            return {
                "cause": f"变量/函数 '{var_name}' 未定义",
                "possible_reasons": [
                    f"拼写错误：检查 '{var_name}' 是否拼写正确",
                    "作用域问题：变量可能在其他作用域定义",
                    "忘记导入：可能需要导入某个模块或函数"
                ],
                "suggestion": f"检查 '{var_name}' 的拼写和作用域"
            }
        return {"cause": "名称错误", "suggestion": "检查变量/函数名"}
        
    def _analyze_type_error(self, error_msg, code, error_location):
        if "unsupported operand type" in error_msg:
            return {
                "cause": "操作数类型不匹配",
                "suggestion": "检查参与运算的变量类型"
            }
        elif "must be str, not" in error_msg:
            return {
                "cause": "类型转换错误",
                "suggestion": "使用str()显式转换为字符串"
            }
        elif "takes exactly" in error_msg:
            return {
                "cause": "函数参数数量错误",
                "suggestion": "检查函数参数个数"
            }
        return {"cause": "类型错误", "suggestion": "检查变量类型"}
        
    def _analyze_index_error(self, error_msg, code, error_location):
        return {
            "cause": "索引越界或访问空序列",
            "possible_reasons": [
                "索引大于等于列表长度",
                "访问空列表/字符串的元素",
                "索引是负数但绝对值太大"
            ],
            "suggestion": "检查索引范围和列表是否为空"
        }
        
    def _analyze_key_error(self, error_msg, code, error_location):
        match = re.search(r"KeyError: '(.+?)'", error_msg)
        if match:
            key = match.group(1)
            return {
                "cause": f"字典中不存在键 '{key}'",
                "suggestion": f"使用 get() 方法或先检查键是否存在：dict.get('{key}', default)"
            }
        return {"cause": "键错误", "suggestion": "检查字典键是否存在"}
        
    def _analyze_attribute_error(self, error_msg, code, error_location):
        match = re.search(r"'(.+?)' object has no attribute '(.+?)'", error_msg)
        if match:
            obj_type, attr = match.groups()
            return {
                "cause": f"'{obj_type}' 类型对象没有 '{attr}' 属性/方法",
                "suggestion": f"检查对象类型和属性/方法名拼写"
            }
        return {"cause": "属性错误", "suggestion": "检查对象类型和属性名"}
        
    def _analyze_syntax_error(self, error_msg, code, error_location):
        return {
            "cause": "语法错误",
            "possible_reasons": [
                "缺少冒号 :",
                "括号/引号不匹配",
                "缩进错误",
                "关键字拼写错误"
            ],
            "suggestion": "检查语法"
        }
        
    def _analyze_generic_error(self, error_type, error_msg, code, error_location):
        return {
            "cause": f"{error_type}: {error_msg}",
            "suggestion": "检查代码逻辑"
        }
```

## 四、实战练习

### 练习1：分析并修复以下错误

**代码：**
```python
def calculate_average(numbers):
    total = sum(numbers)
    count = len(numbers)
    return total / count

# 测试
result = calculate_average([])
print(result)
```

**错误信息：**
```
Traceback (most recent call last):
  File "test.py", line 7, in &lt;module&gt;
    result = calculate_average([])
  File "test.py", line 4, in calculate_average
    return total / count
ZeroDivisionError: division by zero
```

**分析和修复：**
- 错误类型：ZeroDivisionError
- 原因：空列表时count=0
- 修复：添加空列表检查

---

### 练习2：分析并修复以下错误

**代码：**
```python
def get_user_info(user_id):
    users = {
        1: {"name": "张三", "age": 20},
        2: {"name": "李四", "age": 25}
    }
    return users[user_id]

# 测试
info = get_user_info(3)
print(info)
```

**错误信息：**
```
Traceback (most recent call last):
  File "test.py", line 10, in &lt;module&gt;
    info = get_user_info(3)
  File "test.py", line 7, in get_user_info
    return users[user_id]
KeyError: 3
```

**分析和修复：**
- 错误类型：KeyError
- 原因：user_id=3不存在
- 修复：使用get()方法或检查键是否存在
