## 📚 本日核心知识点

### 代码解释器完整闭环 🔄
```
用户需求 → 代码生成 → 沙箱执行 → 结果分析
                              ↓
                    成功？ → 返回结果
                        ↓ 否
                    报错分析 → 自动修正 → 重新执行（最多N次）
```

### 关键概念速查
| 概念 | 一句话说明 |
|------|-----------|
| **代码生成** | 用LLM把自然语言需求转换成可执行代码 |
| **执行沙箱** | 安全执行代码的隔离环境，防止恶意操作 |
| **报错分析** | 理解错误信息，定位问题所在 |
| **自动修正** | 根据报错信息自动修复代码，重新执行 |
| **反思回路** | 分析失败原因，调整策略，迭代优化 |
| **AST** | Abstract Syntax Tree，代码的抽象语法树表示 |
