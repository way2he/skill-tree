## 📚 本日核心知识点

### 大项目收尾和优化全景

```
功能测试
    ├─ 单元测试 (Unit Tests)
    ├─ 集成测试 (Integration Tests)
    └─ 端到端测试 (E2E Tests)

异常处理
    ├─ 错误捕获与分类
    ├─ 重试机制 (Retry)
    ├─ 熔断器 (Circuit Breaker)
    └─ 降级方案 (Fallback)

性能优化
    ├─ 缓存策略 (Caching)
    ├─ 异步处理 (Async)
    ├─ 并发优化 (Concurrency)
    └─ 数据库优化

RAGAS评估
    ├─ 答案相关性 (Answer Relevancy)
    ├─ 上下文相关性 (Context Relevancy)
    ├─ 忠实度 (Faithfulness)
    └─ 上下文召回 (Context Recall)

A/B测试
    ├─ 实验设计
    ├─ 指标选择
    ├─ 流量分配
    └─ 统计分析
```
