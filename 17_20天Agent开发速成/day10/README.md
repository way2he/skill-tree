## 📚 本日核心知识点

### 三层记忆架构
```
感觉记忆 → 短期/工作记忆 → 长期记忆
  (秒级)      (分钟级)       (永久)
```

### 检索策略
```
向量相似度 + 时间衰减 + 重要性评分 → 混合排序
```
