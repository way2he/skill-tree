## 一、记忆系统概述

### 1.1 为什么需要记忆？

```
没有记忆的 Agent = 每次都是第一次
```

**记忆的作用**：
1. 保存历史对话和经验
2. 避免重复提问和错误
3. 实现个性化和上下文理解
4. 支持长期学习和成长

## 二、三层记忆架构

### 2.1 整体架构

```
┌─────────────────────────────────────────┐
│         感觉记忆 (Sensory Memory)       │
│           秒级，暂存输入                 │
└──────────────┬──────────────────────────┘
               ↓
┌─────────────────────────────────────────┐
│      短期/工作记忆 (Short-term Memory)   │
│           分钟级，当前会话               │
└──────────────┬──────────────────────────┘
               ↓
┌─────────────────────────────────────────┐
│         长期记忆 (Long-term Memory)      │
│           永久保存，知识库               │
└─────────────────────────────────────────┘
```

---

### 2.2 各层详细说明

#### 感觉记忆
- **持续时间**：秒级
- **容量**：所有输入
- **作用**：暂存输入，过滤无用信息
- **示例**：用户输入的原始文本

#### 短期/工作记忆
- **持续时间**：会话级（分钟-小时）
- **容量**：当前会话的上下文
- **作用**：保存当前对话状态
- **示例**：对话历史、当前任务状态

#### 长期记忆
- **持续时间**：永久
- **容量**：所有历史
- **作用**：保存知识、经验、偏好
- **示例**：用户资料、历史对话摘要

## 三、实现方案

### 3.1 短期记忆实现

```python
class ShortTermMemory:
    def __init__(self, max_items=50):
        self.items = []  # FIFO 队列
        self.max_items = max_items
    
    def add(self, item):
        self.items.append(item)
        if len(self.items) > self.max_items:
            self.items.pop(0)
    
    def get_recent(self, n):
        return self.items[-n:]
```

### 3.2 长期记忆实现

```python
class LongTermMemory:
    def __init__(self):
        self.vector_store = VectorStore()  # 向量存储
        self.metadata_store = MetadataStore()  # 元数据
    
    def add(self, memory, embedding):
        self.vector_store.add(embedding, memory)
        self.metadata_store.add(memory.id, {
            "timestamp": time.now(),
            "importance": calculate_importance(memory),
            "access_count": 0
        })
    
    def retrieve(self, query, top_k=5):
        # 检索 + 重排序
        candidates = self.vector_store.search(query, top_k*2)
        return rerank(candidates, query)
```

## 四、设计要点

### 4.1 记忆写入路径

```
输入 → 感觉记忆 → 筛选 → 工作记忆 → 编码 → 长期记忆
```

```
查询 → 工作记忆搜索 → 长期记忆检索 → 合并 → 返回
```
