## 核心要点

### 推理加速
| 技术 | 效果 |
|------|------|
| KV Cache | 2-5x 加速 |
| INT8 量化 | 2-3x 加速，显存降 50% |
| INT4 量化 | 3-4x 加速，显存降 75% |
| vLLM | 高吞吐推理 |

### LoRA
- r (Rank) 8-64
- 只训练 0.1-1% 的参数
- LoRA adapter 只有几十 MB
- QLoRA = 量化 + LoRA，更省显存
