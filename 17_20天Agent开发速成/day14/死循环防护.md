# 死循环防护 🔄

## 什么是死循环？

Agent 陷入无限重复的循环，无法前进：
```
A → B → A → B → A → B...
```

## 常见原因

| 原因 | 说明 |
|------|------|
| **相互等待** | Agent A 等 B，B 等 A |
| **任务循环** | 任务 1 触发任务 2，任务 2 又触发任务 1 |
| **条件不满足** | 退出条件永远达不到 |
| **资源死锁** | 互相等待对方释放资源 |

## 防护策略

### 1. 超时机制 ⏰

```python
import time

start_time = time.time()
timeout = 60  # 60秒超时

while True:
    if time.time() - start_time &gt; timeout:
        raise TimeoutError("任务执行超时")
    # 执行任务
```

### 2. 最大步数限制 🚶

```python
max_steps = 10
current_step = 0

while current_step &lt; max_steps:
    # 执行任务
    current_step += 1

if current_step &gt;= max_steps:
    print("已达最大步数，退出")
```

### 3. 重复检测 🔍

```python
history = set()

while True:
    state = get_current_state()

    if state in history:
        print("检测到重复状态，可能陷入循环")
        break

    history.add(state)
    # 执行任务
```

### 4. 心跳检测 ❤️

```python
last_progress_time = time.time()

while True:
    if has_progress():
        last_progress_time = time.time()
    else:
        if time.time() - last_progress_time &gt; 30:
            print("30秒没有进展，退出")
            break
```
