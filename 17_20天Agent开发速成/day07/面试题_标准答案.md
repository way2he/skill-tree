## 第 1 题答案

### 问题
LangChain 和 LangGraph 的区别是什么？为什么要用 LangGraph？

### 标准答案

| 维度 | LangChain | LangGraph |
|------|-----------|-----------|
| **范式** | Chain（链式调用） | Graph（图/状态机） |
| **循环** | 困难，容易无限循环 | 原生支持，可控 |
| **状态** | 隐式，容易混乱 | 显式，State 管理 |
| **分支** | 复杂 | 原生支持条件跳转 |
| **可观测性** | 一般 | 好，每个节点状态都看得到 |
| **适用场景** | 简单线性流程 | 复杂 Agent 流程 |

**一句话**：LangChain 适合简单线性流程，LangGraph 适合复杂 Agent（需要循环、分支、显式状态）。

## 第 2 题答案

### 问题
梯度下降三种变体（SGD、Momentum、Adam）的区别？

### 标准答案

| 变体 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| **SGD** | 每次用 1 个样本更新 | 快，内存小 | 不稳定，震荡 |
| **Momentum** | 加入惯性，累计之前的梯度（β≈0.9） | 减少震荡，收敛快 | 超参数 β |
| **Adam** | 自适应学习率，Momentum + RMSProp | 收敛快，稳 | 计算稍复杂 |

## 第 3 题答案

### 问题
Transformer 的完整架构是什么？请详细说明每个组件。

### 标准答案

#### 核心组件列表

| 组件 | 作用 |
|------|------|
| **Embedding + Position Encoding** | 词向量 + 位置信息 |
| **Multi-Head Self-Attention** | 多头自注意力，关注不同子空间 |
| **LayerNorm + Residual** | 层归一化 + 残差连接 |
| **Feed Forward Network (FFN)** | 前馈网络，非线性变换 |
| **Cross-Attention（Decoder）** | 解码器关注编码器输出 |

#### Encoder-only / Decoder-only / Encoder-Decoder

| 架构 | 代表 | 适用任务 |
|------|------|---------|
| Encoder-only | BERT/ERNIE | 理解类（分类、NER、QA） |
| Decoder-only | GPT/Llama | 生成类（文本生成） |
| Encoder-Decoder | T5/BART | 翻译、摘要 |

## 第 4 题答案

### 问题
为什么 Transformer 用 LayerNorm 而不是 BatchNorm？

### 标准答案（3 个原因）

| 原因 | 说明 |
|------|------|
| **序列长度可变** | LayerNorm 在每个样本内部归一化，不受序列长度影响；BatchNorm 在 batch 内归一化，需要长度一致 |
| **Batch size 小** | LayerNorm 对 batch size 不敏感，BatchNorm batch size 小效果差 |
| **推理稳定** | LayerNorm 每个样本独立，推理时直接用；BatchNorm 需要保存 running mean/var |

## 第 5 题答案

### 问题
Self-Attention 的公式推导和计算过程？

### 标准答案

#### 公式（面试必背！）
```
Attention(Q, K, V) = softmax(Q·K^T / √d_k) · V
```

#### Q/K/V 说明
- Q (Query)：当前词"问"其他词
- K (Key)：其他词的"钥匙"
- V (Value)：其他词的"值"

#### 为什么除以 √d_k
- 防止 Q·K^T 太大，softmax 梯度消失
