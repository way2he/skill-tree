## LangGraph 四大核心概念快速记忆

| 概念 | 一句话 |
|------|--------|
| **State** | Agent 的记忆（messages/step/is_done） |
| **Node** | 执行单元（纯函数优先） |
| **Edge** | 连接节点（条件边很强大） |
| **Graph** | 完整流程（State + Nodes + Edges） |

## Transformer 面试必背 5 点

1. **整体架构**：Embedding + PE → Encoder Stack → Decoder Stack → Output
2. **Self-Attention 公式**：Attention(Q,K,V) = softmax(QK^T/√d_k)V
3. **为什么用 LayerNorm**：3 个原因（序列长度、batch size、推理稳定）
4. **三个分支**：Encoder-only（BERT）/ Decoder-only（GPT）/ Encoder-Decoder（T5）
5. **核心组件作用**：Multi-Head、Residual、FFN
