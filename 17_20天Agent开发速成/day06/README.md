## 📚 今日核心知识点

### RAG 优化方法 🔧
1. **分块策略**：固定大小、语义分块、父子分块
2. **元数据过滤**：按时间、分类、作者等过滤
3. **混合检索**：BM25（关键词）+ 向量（语义）
4. **查询重写**：让用户查询更适合检索
5. **Rerank**：精排提升准确率（Day05 已学）

### RAGAS 评估指标 📊
1. **Faithfulness**：答案是否忠实于检索到的上下文
2. **Answer Relevance**：答案是否相关
3. **Context Precision**：上下文精准度
4. **Context Recall**：上下文召回率
5. **Context Relevance**：上下文相关性

### NLP 任务 📝
- **文本分类**：把文本分到预定义类别
- **NER（命名实体识别）**：识别文本中的实体（人名、地名、组织等）
