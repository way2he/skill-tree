## 第一部分：文本分类

### 什么是文本分类？

把文本分到预定义的类别里。

### 常见应用

| 应用 | 说明 |
|------|------|
| **情感分析** | 好评/中评/差评 |
| **垃圾邮件检测** | 垃圾邮件/正常邮件 |
| **新闻分类** | 科技/体育/娱乐/财经 |
| **意图识别** | 客服场景，用户想干什么 |

---

### 文本分类方法演进

#### 1. 传统 ML 方法（2018 年以前）

| 方法 | 说明 |
|------|------|
| **TF-IDF + SVM** | 经典组合 |
| **TF-IDF + Logistic Regression** | 简单快速 |
| **Naive Bayes** | 朴素贝叶斯 |
| **FastText** | Facebook 的方法，快 |

**流程**
```
文本 → TF-IDF/词袋 → 训练分类器（SVM/LR） → 预测
```

---

#### 2. 深度学习方法（2018 年以后）

| 方法 | 说明 |
|------|------|
| **BERT/ERNIE 等预训练模型** | 效果最好 |
| **TextCNN** | CNN 做文本分类 |
| **BiLSTM/GRU** | 循环神经网络 |

**流程**
```
文本 → BERT Tokenizer → BERT 编码 → 分类头 → 预测
```

---

### 传统 ML vs 深度学习对比

| 维度 | 传统 ML | 深度学习 |
|------|---------|---------|
| **实现难度** | ⭐ 简单 | ⭐⭐⭐ 中等 |
| **效果** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **数据要求** | 少 | 较多 |
| **计算资源** | 少 | 多 |
| **可解释性** | 高 | 低 |

## 第二部分：NER（命名实体识别）

### 什么是 NER？

识别文本中的**命名实体**。

### 常见实体类型

| 实体类型 | 说明 | 例子 |
|---------|------|------|
| **人名** | Person | 张三、Elon Musk |
| **地名** | Location | 北京、New York |
| **组织名** | Organization | 阿里巴巴、Google |
| **时间** | Time | 2024-01-15、明天 |
| **其他** | 自定义 | 产品名、型号等 |

---

### NER 例子

```
输入: "张三在 2024 年 1 月去北京的阿里巴巴出差。"

输出:
- 张三: PERSON
- 2024 年 1 月: TIME
- 北京: LOCATION
- 阿里巴巴: ORGANIZATION
```

---

### NER 常用方法

| 方法 | 说明 |
|------|------|
| **规则/词典** | 简单场景用，维护成本高 |
| **CRF（条件随机场）** | 传统方法，效果不错 |
| **BiLSTM-CRF** | 深度学习经典组合 |
| **BERT/ERNIE + 分类头** | 效果最好 |

---

### NER 实战（HuggingFace）

```python
# -*- coding: utf-8 -*-
"""
NER 实战示例（HuggingFace）
"""

print("=" * 60)
print("NER 实战示例")
print("=" * 60)

text = "张三在 2024 年 1 月去北京的阿里巴巴出差。"
print(f"\n输入文本: {text}")

# 模拟 NER 输出
print("\nNER 输出:")
print("  [人名] 张三")
print("  [时间] 2024 年 1 月")
print("  [地名] 北京")
print("  [组织名] 阿里巴巴")

print("\n" + "=" * 60)
print("NER 常用方法")
print("=" * 60)
print("""
1. 规则/词典: 简单场景
2. CRF: 传统方法，效果不错
3. BiLSTM-CRF: 深度学习经典
4. BERT/ERNIE: 效果最好
""")
```

## 本讲小结 ✅

### 文本分类
- **传统 ML**：TF-IDF + SVM/LR
- **深度学习**：BERT/TextCNN/BiLSTM

### NER
- **任务**：识别实体（人名/地名/组织名/时间）
- **方法**：CRF、BiLSTM-CRF、BERT
