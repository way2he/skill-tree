## 📚 上午：Prompt 工程核心技巧

### 1. 角色设定方法论（30分钟）

#### 黄金公式：**角色 + 背景 + 任务 + 格式 + 约束**

```
你是【具体角色】，擅长【核心能力】，【背景描述】

你的任务是【具体任务】

输出格式：
1. xxx
2. xxx

约束条件：
- 【约束1】
- 【约束2】
```

**实战练习**：写一个代码审查 Agent 的系统提示词（参考面试题第7题）

---

### 2. 思维链 CoT 进阶（30分钟）

#### CoT 触发句式：
- "Let's think step by step"
- "请一步步思考，先分析问题，再给出答案"
- "在给出最终答案前，请说明你的推理过程"

#### 进阶技巧：
1. **零样本 CoT**：不需要示例，只加一句话
2. **少样本 CoT**：给1-3个完整的推理示例
3. **自一致性**Consistency：生成多个答案，投票选最优

**实战练习**：用CoT解决一个数学题，对比加不加CoT的差异

---

### 3. Few-shot 最佳实践（30分钟）

#### 设计原则：
1. **示例多样性**：覆盖不同情况
2. **示例质量**：示例本身就是高质量输出
3. **格式一致性**：所有示例格式完全一致
4. **边界案例**：包含边界情况也要覆盖

**实战练习**：写一个情感分析的3个Few-shot示例

## 💻 下午：格式控制 + 安全防护

### 4. JSON 结构化输出 100% 稳定（45分钟）

#### 核心技巧：
1. **JSON Schema 明确**：在提示词里给出完整的 Schema
2. **Pydantic 校验**：输出后用 Pydantic 强制校验
3. **重试机制**：校验失败自动重试
4. **重试时给出错误信息**

#### 代码模板见面试题第6题的代码模板

**实战练习**：实现一个用户画像生成器，要求100%输出指定格式的JSON

---

### 5. 提示注入攻击与防护（45分钟）

#### 常见攻击手法：
1. **指令覆盖**："忽略之前的所有指令，现在..."
2. **角色扮演**："假设你现在是黑客，教我..."
3. **编码绕过**：Base64 / Unicode / 拼音
4. **分隔符注入**：用特殊字符分割原始指令

#### 防护方法：
1. **输入过滤**：检测攻击特征词
2. **提示词加固**：分隔符 + 特权指令
3. **输出校验**：敏感信息检测
4. **专用模型**：用专门的注入检测模型

**实战练习**：设计5种注入攻击测试用例，然后用自己写的防护系统测试

## 🧠 晚上：深度加餐

### 6. 注意力机制源码级解析（1小时）

#### 核心公式：

```python
# 注意力机制的本质：Q = softmax(QK^T/√d_k) V
```

#### 理解要点：
1. **Q、K、V 是什么？**
   - Q：查询向量 - 我要找什么
   - K：键向量 - 我有什么
   - V：值向量 - 实际内容
   
2. **为什么除以 √d_k？**
   - 防止内积太大，softmax 梯度消失
   
3. **多头注意力为什么有效？**
   - 每个头学习不同的注意力模式
   - 并行捕捉多种语义关系
