1. **第8题是开放题，一定要结合正则化思想答！** 能说出来L1/L2/Dropout/早停直接加分，说明你不是只会调Prompt的调参侠，你有理论基础！
2. **所有问题尽量结合 Agent 开发场景**：比如 JSON 输出在 Agent 调用工具、调用函数的时候100%要用到！
3. **有量化数据最好**：比如"加上Pydantic结构化输出+2次重试，JSON格式成功率从80%提升到99.9%"
4. **提示注入一定要说清楚5种攻击手法+4种防护**，展示你有安全意识！
