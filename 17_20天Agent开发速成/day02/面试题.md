**问题**：请解释 Few-shot learning、One-shot learning、Zero-shot learning 的区别，并各举一个应用场景。

---

### 第3题：提示注入攻击（8分）
**问题**：什么是提示注入攻击？请列举 3 种常见的提示注入手法以及对应的防护方法。

---

### 第4题：注意力机制本质（8分）
**问题**：请用一句话解释注意力机制的本质，然后说明 Q、K、V 三个向量分别代表什么。

---

### 第5题：多头注意力（8分）
**问题**：为什么要用「多头」注意力？单头注意力有什么局限性？

## 二、代码题（共40分，每题20分）

### 第6题：JSON 格式强制输出（20分）
**问题**：请设计一个系统提示词 + 输出校验函数，要求大模型 100% 输出如下格式的 JSON：

```json
{
  "review_result": "approve" | "reject",
  "confidence": 0.0-1.0,
  "comments": "string",
  "suggestions": ["string", "string"]
}
```

要求：
1. 写出完整的系统提示词
2. 实现一个 Python 校验函数，验证输出是否符合格式
3. 如果校验失败，自动重试最多 3 次

---

### 第7题：CoT 提示词优化（20分）
**问题**：下面是一个写得不好的提示词，请优化它，要求加入：
1. 清晰的角色设定
2. 思维链（CoT）要求
3. 结构化输出格式
4. Few-shot 示例（1-2个）

**原始糟糕提示词**：
```
帮我算一下这个问题：商店里有苹果和香蕉，苹果的数量是香蕉的2倍，总共有15个水果，问苹果有几个？
```
