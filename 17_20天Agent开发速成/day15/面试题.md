# Day15 面试题（6 道）📝

## 基础题（3 道）

### 第 1 题
请为代码解释器 Agent 写一份完整的 PRD（产品需求文档）。

### 第 2 题
如何设计安全的代码执行环境？有哪些安全风险需要考虑？

### 第 3 题
如何评估代码解释器的效果？有什么指标？

---

## 进阶题（3 道）

### 第 4 题
请画出代码解释器 Agent 的完整系统架构图，并解释每个模块的职责。

### 第 5 题
在代码执行时，如何优雅地处理错误？请设计一个错误处理和修复流程。

### 第 6 题
如何让 Agent 能使用工具（搜索、文件操作、API 等）？请设计工具系统的架构。
