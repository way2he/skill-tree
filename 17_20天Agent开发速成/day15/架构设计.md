# 架构设计 🏗️

## 系统架构图

```
┌─────────────────────────────────────────────────────────────┐
│                        用户界面层                              │
│                    (Web / CLI / API)                         │
└────────────────────────────┬────────────────────────────────┘
                             │
┌────────────────────────────▼────────────────────────────────┐
│                     协调层 (Orchestrator)                      │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  1. 意图理解                                         │   │
│  │  2. 任务分解                                         │   │
│  │  3. 流程控制                                         │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────┬──────────────────┬────────────────────┘
                     │                  │
        ┌────────────┴──────┬───────────┴────────────┐
        │                   │                        │
┌───────▼───────┐  ┌──────▼───────┐   ┌────────────▼──────────┐
│  代码生成层   │  │  代码执行层  │   │  结果解释层           │
│               │  │              │   │                       │
│ - Prompt 设计 │  │ - 沙箱环境  │   │ - 结果分析           │
│ - 代码生成    │  │ - 资源限制  │   │ - 自然语言解释     │
│ - 代码检查    │  │ - 安全检查  │   │ - 可视化             │
└───────────────┘  └──────────────┘   └───────────────────────┘
        │                   │                        │
        └────────────┬──────┴───────────┬────────────┘
                     │                  │
┌────────────────────▼──────────────────▼────────────────────┐
│                        工具层 (Tools)                        │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐       │
│  │ 文件操作 │ │ 数据处理 │ │ 网络搜索 │ │ API调用  │       │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘       │
└────────────────────────────────────────────────────────────┘
```

## 核心模块

| 模块 | 职责 |
|------|------|
| **Orchestrator** | 协调各个模块，控制流程 |
| **CodeGenerator** | 生成代码 |
| **CodeExecutor** | 执行代码 |
| **ResultInterpreter** | 解释结果 |
| **ToolManager** | 管理工具 |
