## 项目目标

我们要做一个 **代码解释器 Agent**：

```
用户输入自然语言描述任务
  ↓
Agent 理解意图
  ↓
Agent 写代码
  ↓
Agent 执行代码
  ↓
Agent 解释结果
  ↓
输出给用户
```

### 示例场景

- 📊 **数据分析**: "帮我分析这个 CSV 文件，画个趋势图"
- 🔢 **数学计算**: "解这个微分方程"
- 📈 **可视化**: "用 matplotlib 画个直方图"
- 🛠️ **自动化**: "写个脚本帮我批量重命名文件"

---

## 核心功能

| 模块 | 功能 |
|------|------|
| 意图理解 | 理解用户想做什么 |
| 代码生成 | 生成合适的代码 |
| 代码执行 | 安全地执行代码 |
| 结果解释 | 把结果用自然语言解释 |
| 错误处理 | 出错时自动修复 |
| 多轮对话 | 支持持续的对话 |
