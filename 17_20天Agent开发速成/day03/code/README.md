## 每个代码文件的学习要点

### 📌 01_stream_chat.py
- 理解 `stream=True` 参数的作用
- 观察流式输出逐字打印的用户体验
- 思考什么时候用流式，什么时候用非流式
- 面试时能说出 SSE 技术的原理和好处

### 📌 02_retry_decorator.py
- 理解指数退避的数学原理（1s → 2s → 4s → 8s...）
- 重点理解为什么要加 `jitter`（随机抖动）
- 理解惊群效应是什么，怎么避免
- 装饰器的写法，`@wraps` 的作用

### 📌 03_token_counter.py
- 理解 tiktoken 的使用方法
- 记住 1K Token ≈ 750 英文单词 ≈ 500 中文汉字
- 理解消息计数的固定开销（每轮 3 个 Token）
- 会计算一次 API 调用的大致成本

### 📌 04_unified_llm_interface.py
- 理解为什么要设计统一接口层（开闭原则）
- 理解策略模式：每个模型一个策略类
- 理解工厂模式：根据配置创建实例
- 理解适配器模式：统一不同 SDK 的接口
- 体会「换模型只改一行配置」的爽点

### 📌 05_fallback_strategy.py
- 理解高可用系统的设计思想
- 理解熔断机制：连续失败几次就暂时禁用
- 理解优先级排序：保证效果最好的模型优先
- 理解故障转移：主节点挂了自动切备用节点
