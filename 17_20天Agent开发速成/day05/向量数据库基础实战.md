## 什么是向量数据库？

### 传统数据库 vs 向量数据库

| 特性 | 传统数据库（MySQL/PostgreSQL） | 向量数据库（Chroma/Qdrant） |
|------|------------------------------|---------------------------|
| **存的是什么** | 表格、JSON、文档 | 高维向量 |
| **怎么查** | `WHERE name = "张三"`（精确匹配） | `WHERE similar(vector, [0.1, 0.2, ...])`（相似度检索） |
| **索引方式** | B-Tree, Hash | HNSW, IVF, FAISS |
| **适用场景** | 精确查询 | 语义搜索、RAG |

---

## 主流向量数据库对比

| 数据库 | 类型 | 部署 | 速度 | 易用性 | 推荐场景 |
|--------|------|------|------|--------|---------|
| **Chroma** | 本地 | Python 包 | ⚡⚡⚡ | ⭐⭐⭐⭐⭐ | 开发、原型 |
| **Qdrant** | 本地/服务 | Docker/Python | ⚡⚡⚡⚡ | ⭐⭐⭐⭐ | 生产、中小规模 |
| **Pinecone** | 云服务 | SaaS | ⚡⚡⚡⚡⚡ | ⭐⭐⭐ | 生产、大规模 |
| **Weaviate** | 本地/服务 | Docker | ⚡⚡⚡ | ⭐⭐⭐ | 图+向量 |
| **Milvus** | 本地/服务 | Docker/K8s | ⚡⚡⚡⚡ | ⭐⭐ | 生产、超大规模 |

## 实战一：Chroma 快速入门 🔥

### 1. 安装
```bash
pip install chromadb sentence-transformers
```

### 2. 最简示例
```python
# -*- coding: utf-8 -*-
import chromadb
from chromadb.utils import embedding_functions

# 1. 初始化客户端（本地存储）
client = chromadb.PersistentClient(path="./chroma_db")

# 2. 创建集合（相当于表）
collection = client.create_collection(
    name="my_documents",
    metadata={"description": "我的文档集合"}
)

# 3. 准备文档
documents = [
    "Python 是一种解释型编程语言",
    "JavaScript 主要用于网页开发",
    "Java 是编译型语言，运行在 JVM 上",
    "Go 语言擅长高并发场景",
    "Rust 注重内存安全"
]

metadatas = [
    {"category": "programming", "year": 1991},
    {"category": "programming", "year": 1995},
    {"category": "programming", "year": 1995},
    {"category": "programming", "year": 2009},
    {"category": "programming", "year": 2010}
]

ids = ["doc1", "doc2", "doc3", "doc4", "doc5"]

# 4. 添加文档（自动用 Sentence-BERT 向量化）
collection.add(
    documents=documents,
    metadatas=metadatas,
    ids=ids
)

print(f"✅ 添加了 {len(documents)} 个文档")

# 5. 查询
query_text = "什么语言适合做高并发？"
results = collection.query(
    query_texts=[query_text],
    n_results=3  # 返回 Top 3
)

print("\n" + "=" * 50)
print(f"查询: {query_text}")
print("=" * 50)

for i in range(len(results['ids'][0])):
    print(f"\n--- 结果 {i+1} ---")
    print(f"ID: {results['ids'][0][i]}")
    print(f"距离: {results['distances'][0][i]:.4f}")
    print(f"内容: {results['documents'][0][i]}")
    print(f"元数据: {results['metadatas'][0][i]}")
```

### 3. 常用操作
```python
# 查看所有集合
print(client.list_collections())

# 获取已有的集合
collection = client.get_collection(name="my_documents")

# 更新文档
collection.update(
    ids=["doc1"],
    documents=["Python 是一种解释型、高级、通用的编程语言"]
)

# 删除文档
collection.delete(ids=["doc5"])

# 删除集合
client.delete_collection(name="my_documents")
```

### 4. 用自己的 Embedding 模型
```python
from sentence_transformers import SentenceTransformer

model = SentenceTransformer('BAAI/bge-small-zh-v1.5')

# 自定义 Embedding 函数
class MyEmbeddingFunction:
    def __call__(self, input):
        return model.encode(input).tolist()

# 创建集合时指定
collection = client.create_collection(
    name="my_collection",
    embedding_function=MyEmbeddingFunction()
)
```

## 实战二：Qdrant 快速入门 🚀

### 1. 启动 Qdrant（Docker）
```bash
docker pull qdrant/qdrant
docker run -p 6333:6333 -v $(pwd)/qdrant_storage:/qdrant/storage qdrant/qdrant
```

Windows (PowerShell):
```powershell
docker run -p 6333:6333 -v ${PWD}/qdrant_storage:/qdrant/storage qdrant/qdrant
```

### 2. Python 客户端示例
```python
# -*- coding: utf-8 -*-
from qdrant_client import QdrantClient
from qdrant_client.http import models
from sentence_transformers import SentenceTransformer

# 1. 初始化客户端
client = QdrantClient(host="localhost", port=6333)

# 2. 加载 Embedding 模型
model = SentenceTransformer('BAAI/bge-small-zh-v1.5')

# 3. 创建集合
collection_name = "my_documents"
vector_size = 512  # BGE-small 的维度

client.recreate_collection(
    collection_name=collection_name,
    vectors_config=models.VectorParams(
        size=vector_size,
        distance=models.Distance.COSINE  # 余弦相似度
    )
)

# 4. 准备文档
documents = [
    "Python 是一种解释型编程语言",
    "JavaScript 主要用于网页开发",
    "Java 是编译型语言，运行在 JVM 上",
    "Go 语言擅长高并发场景",
    "Rust 注重内存安全"
]

# 5. 向量化
vectors = model.encode(documents).tolist()

# 6. 插入数据
points = [
    models.PointStruct(
        id=i,
        vector=vectors[i],
        payload={
            "text": documents[i],
            "category": "programming"
        }
    )
    for i in range(len(documents))
]

client.upsert(
    collection_name=collection_name,
    points=points
)

print(f"✅ 插入了 {len(documents)} 个点")

# 7. 查询
query_text = "什么语言适合做高并发？"
query_vector = model.encode(query_text).tolist()

search_result = client.search(
    collection_name=collection_name,
    query_vector=query_vector,
    limit=3
)

print("\n" + "=" * 50)
print(f"查询: {query_text}")
print("=" * 50)

for i, hit in enumerate(search_result, 1):
    print(f"\n--- 结果 {i} ---")
    print(f"ID: {hit.id}")
    print(f"分数: {hit.score:.4f}")
    print(f"内容: {hit.payload['text']}")
```

## Chroma vs Qdrant 对比

| 维度 | Chroma | Qdrant |
|------|--------|--------|
| **易用性** | ⭐⭐⭐⭐⭐（最简单） | ⭐⭐⭐⭐ |
| **性能** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **功能** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **部署** | pip install | Docker |
| **元数据过滤** | ✅ | ✅✅（更强大） |
| **推荐用途** | 开发、原型 | 生产环境 |

### 选型建议
- **快速原型**：Chroma
- **生产环境**：Qdrant
- **超大规模**：Pinecone/Milvus
