# 命令类型
| 类型 | 目录位置 | 调用方式 | 作用范围 |
|------|----------|----------|----------|
| 项目级 | .claude/commands/ | /project:<name> | 仅当前项目 |
| 用户级 | ~/.claude/commands/ | /user:<name> | 所有项目 |

# 创建项目级命令

在终端中，先 cd 到项目根目录
1.  创建命令目录 
> mkdir -p .claude\commands 
2. 创建优化命令文件
> touch .claude/commands/optimize.md
> Set-Content -Path ".claude\commands\optimize.md" -Value "分析这个项目的性能，并提出三个具体的优化建议。
3. notepad .claude\commands\optimize.md
4. 使用方式
> /project:optimize 

# 创建用户级命令
在终端中（任意目录）
```bash
# 创建用户级命令目录                                               │
│ PS C:\> mkdir -p $HOME\.claude\commands                          │
│                                                                  │
│ # 创建推送命令文件                                                 │
│ PS C:\> Set-Content -Path "$HOME\.claude\commands\push.md" -Value "用合理描述性信息提交所有变更文件，然后推送到远程仓库。" │
│                                                                  │
│ # 或者用记事本编辑                                                 │
│ PS C:\> notepad $HOME\.claude\commands\push.md 

# 使用方式
/user:push
```
