# 核心命令

| 命令 | 功能描述 | 说明 |
|------|----------|------|
| /login | 登录账户 | 首次使用时认证 |
| /init | 扫描项目生成报告 | 快速了解项目结构 |
| /model | 查看/切换模型 | 切换到更强大的模型 |
| /config | 配置设置 | 调整行为偏好 |
| /commit-push-pr | Git 工作流 | 一键提交、推送并创建 PR |

| 命令 | 功能描述 | 说明 |
|------|----------|------|
| `npm install -g @anthropic/claude-code` | 全局安装 Claude Code | 在系统中全局安装 Claude Code CLI 工具 |
| `claude login` | 登录认证 | 登录 Anthropic 账户进行认证 |
| `claude auth status` | 检查认证状态 | 查看当前认证状态 |
| `claude` | 启动交互模式 | 进入 Claude Code 交互式命令行 |

| 命令 | 功能说明 |
|------|----------|
| /help | 显示帮助和可用命令 |
| /init | 初始化项目，生成 CLAUDE.md |
| /clear | 清除对话历史 |
| /compact [instructions] | 压缩对话，可指定保留重点 |
| /exit | 退出 Claude Code |

# 会话相关

# Git 相关

用自然语言操作 Git，无需记忆复杂命令：

| 需求 | 自然语言 |
|------|----------|
| 查看修改 | 我修改了哪些文件 |
| 智能提交 | 用合理描述性信息提交我的更改 |
| 推送代码 | 推送本分支到远程 |
| 切换分支 | 删除本分支并切换到 master 分支 |
| 查看历史 | 显示最近 3 次提交中所有文件列表 |
| 创建 PR | 为这些更改创建一个 Pull Request |
| 解决冲突 | 帮我解决合并冲突 |
