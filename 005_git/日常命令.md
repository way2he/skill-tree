# 最常用的 Git 命令

## 1. 基础配置命令
- `git config --global user.name "Your Name"`：设置全局用户名
- `git config --global user.email "your.email@example.com"`：设置全局邮箱
- `git config --list`：查看所有 Git 配置

## 2. 仓库操作命令
- `git init`：初始化一个新的 Git 仓库
- `git clone [url]`：克隆远程仓库到本地
- `git remote add origin [url]`：关联本地仓库与远程仓库
- `git remote -v`：查看远程仓库地址

## 3. 工作区操作命令
- `git status`：查看当前工作区状态
- `git add [file]` 或 `git add .`：将文件添加到暂存区（`.` 表示所有文件）
- `git commit -m "commit message"`：提交暂存区到本地仓库
- `git commit --amend`：修改最近一次提交的信息

## 4. 分支操作命令
- `git branch`：查看所有本地分支
- `git branch [branch-name]`：创建新分支
- `git checkout [branch-name]`：切换到指定分支
- `git checkout -b [new-branch-name]`：创建并切换到新分支
- `git merge [branch-name]`：将指定分支合并到当前分支
- `git branch -d [branch-name]`：删除本地分支
- `git push origin [branch-name]`：将本地分支推送到远程仓库
- `git push -u origin [branch-name]`：将本地分支与远程分支关联并推送

## 5. 远程仓库同步命令
- `git fetch`：从远程仓库获取最新代码但不合并
- `git pull`：从远程仓库获取最新代码并合并到本地
- `git push`：将本地代码推送到远程仓库

## 6. 查看历史记录命令
- `git log`：查看提交历史
- `git log --oneline`：以简洁方式查看提交历史
- `git log --graph`：以图形方式查看分支合并历史
- `git log --author="Your Name"`：查看指定作者的提交历史

## 7. 撤销操作命令
- `git checkout -- [file]`：撤销对文件的修改（未添加到暂存区）
- `git reset HEAD [file]`：将文件从暂存区撤销到工作区
- `git revert [commit-hash]`：创建一个新提交来撤销指定提交的更改
- `git reset --hard [commit-hash]`：将代码库重置到指定提交状态（会丢失未提交的更改）

## 8. 其他有用命令
- `git diff`：查看文件修改内容
- `git stash`：将当前工作区的更改暂存起来
- `git stash pop`：恢复最近一次暂存的更改
- `git tag [tag-name]`：为当前提交创建标签
- `git show [commit-hash/tag-name]`：查看指定提交或标签的详细信息

## 9. remote相关命令
- `git remote`：查看远程仓库信息
- `git remote add [name] [url]`：添加远程仓库
- `git remote remove [name]`：删除远程仓库
- `git remote rename [old-name] [new-name]`：重命名远程仓库
