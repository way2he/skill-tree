# Ubuntu 24.04 中文字体安装指南

本文档提供了在Ubuntu 24.04系统上安装"SimHei"、"WenQuanYi Micro Hei"、"Heiti TC"和"sans-serif"字体的详细步骤。

## 1. 安装WenQuanYi Micro Hei字体

WenQuanYi Micro Hei字体可以通过Ubuntu的官方软件源直接安装：

```bash
# 更新软件包列表
sudo apt update

# 安装WenQuanYi Micro Hei字体
sudo apt install -y ttf-wqy-microhei
```

## 2. 安装SimHei字体

SimHei字体需要手动下载并安装：

```bash
# 创建字体目录
mkdir -p ~/.local/share/fonts/chinese

# 下载SimHei字体
wget -O ~/.local/share/fonts/chinese/simhei.ttf "https://github.com/fangwentong/windows-fonts/raw/master/SimHei.ttf"

# 如果GitHub链接无法访问，尝试备用链接
# wget -O ~/.local/share/fonts/chinese/simhei.ttf "https://gitcode.net/mirrors/fangwentong/windows-fonts/raw/master/SimHei.ttf"
# wget -O ~/.local/share/fonts/chinese/simhei.ttf "https://cdn.jsdelivr.net/npm/windows-fonts@1.0.4/fonts/SimHei.ttf"
```

## 3. 安装Heiti TC字体

Heiti TC字体同样需要手动下载并安装：

```bash
# 下载Heiti TC字体
wget -O ~/.local/share/fonts/chinese/heititc.ttf "https://github.com/StellarCN/scp_assets/raw/master/fonts/HeitiTC.ttf"

# 如果GitHub链接无法访问，尝试备用链接
# wget -O ~/.local/share/fonts/chinese/heititc.ttf "https://gitcode.net/mirrors/StellarCN/scp_assets/raw/master/fonts/HeitiTC.ttf"

# 或者使用Noto Sans TC作为替代
# wget -O ~/.local/share/fonts/chinese/HeitiTC.ttf "https://github.com/googlefonts/noto-cjk/blob/main/Sans/OTF/TraditionalChinese/NotoSansTC-Regular.otf?raw=true"
```

## 4. 更新字体缓存

安装完字体后，需要更新系统字体缓存：

```bash
fc-cache -f -v
```

## 5. 验证安装

使用以下命令验证字体是否安装成功：

```bash
fc-list | grep -E "SimHei|WenQuanYi Micro Hei|Heiti TC|Noto Sans TC"
```

## 6. 关于sans-serif字体

"sans-serif"是一个字体族名称，而不是具体的字体名称。系统会根据这个名称自动选择可用的无衬线字体（如上述安装的字体）。

## 7. 故障排除

- 如果字体下载失败，请尝试使用提供的备用链接
- 确保字体文件权限正确：`chmod 644 ~/.local/share/fonts/chinese/*`
- 如果字体仍未显示，尝试注销并重新登录

希望本指南能帮助您成功安装所需的中文字体！
