# 正规方程

## 作用
正规方程（Normal Equation）是线性回归中用于直接求解最优参数的数学方法。它通过最小化均方误差损失函数，直接计算出使得损失函数最小的模型参数值，无需使用梯度下降等迭代算法。

## 表达式
对于线性回归模型 $y = X\theta + \varepsilon$，正规方程的表达式为：

$$\theta = (X^T X)^{-1} X^T y$$

其中：
- $X$ 是特征矩阵（每行代表一个样本，每列代表一个特征）
- $y$ 是目标变量向量
- $\theta$ 是模型参数向量（待求解）
- $X^T$ 表示矩阵 $X$ 的转置
- $(X^T X)^{-1}$ 表示矩阵 $X^T X$ 的逆矩阵

## 优点
1. **直接求解**：无需迭代过程，一步计算即可得到最优参数
2. **无需调参**：不需要选择学习率等超参数
3. **理论清晰**：有明确的数学解释，结果是全局最优解
4. **计算高效**：当特征数量较小时（通常小于10,000），计算速度快

## 缺点
1. **计算复杂度高**：当特征数量很大时，计算逆矩阵的时间复杂度为 $O(n^3)$，其中 $n$ 是特征数量
2. **内存消耗大**：需要存储整个特征矩阵并计算其转置和逆矩阵
3. **不适用于非满秩矩阵**：当 $X^T X$ 不可逆时（如特征之间存在高度相关性），无法直接求解
4. **扩展性差**：对于大规模数据集，计算效率远低于梯度下降等迭代方法
