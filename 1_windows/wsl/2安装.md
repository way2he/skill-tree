# WSL Ubuntu 安装方案

## 方案一：自动安装（最简单）

```powershell
# 以管理员身份打开PowerShell，执行：
wsl --install
```

这将自动安装WSL和Ubuntu默认版本。

## 方案二：手动选择版本安装

```powershell
# 1. 查看可用的Linux发行版
wsl --list --online

# 2. 安装指定的Ubuntu版本
wsl --install -d Ubuntu-24.04
# 或
wsl --install -d Ubuntu-22.04
# 或
wsl --install -d Ubuntu
```

## 方案三：通过Microsoft Store安装

1. 打开Microsoft Store
2. 搜索"Ubuntu"
3. 选择版本（24.04 LTS / 22.04 LTS）
4. 点击"获取"安装


## 常用WSL命令

```powershell
# 查看已安装的发行版
wsl --list --verbose

# 启动特定发行版
wsl -d Ubuntu-24.04

# 停止WSL
wsl --shutdown

# 设置默认发行版
wsl --set-default Ubuntu-24.04

# 导出发行版（备份）
wsl --export Ubuntu-24.04 D:\backup\ubuntu.tar

# 导入发行版
wsl --import Ubuntu-Backup D:\wsl\Ubuntu-Backup D:\backup\ubuntu.tar
```

## 推荐配置

安装后建议进行：

```bash
# 更新软件包
sudo apt update && sudo apt upgrade -y

# 安装常用工具
sudo apt install -y git curl wget vim zsh

# 配置中文环境（可选）
sudo apt install -y language-pack-zh-hans
```
