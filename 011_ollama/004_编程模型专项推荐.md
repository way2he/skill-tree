## ⚙️ Windows环境运行优化技巧
```powershell
# 显存最大化利用，尽可能多的模型层放到GPU
set OLLAMA_GPU_LAYERS=99
set OLLAMA_GPU_MEMORY_FRACTION=0.95  # 限制最大用15.2GB显存，避免溢出
# 16GB内存缓存，提升长代码加载和处理速度
set OLLAMA_CACHE_SIZE=16GB
```
1. 模型文件存放在NVMe SSD上，加载速度提升3倍
2. 关闭Windows虚拟内存页面文件，避免内存数据频繁交换导致卡顿
3. 优先使用Ollama官方Windows版本，比WSL2运行性能高20%以上

## 🚫 不推荐模型
- 参数量超过40B的编程模型：需要大量offload到内存，速度<10 tokens/s，体验极差
- 老旧的第一代代码模型（如code-llama:7b-v1）：能力远不如新一代模型，没必要使用
