## ⚖️ 评测维度说明
共9个核心评测维度，满分10分：
1. **显存适配性 (20%)**：针对16G显存的适配程度，是否需要offload到内存
2. **推理速度 (15%)**：RTX5080上每秒生成token数，首包响应速度
3. **中文能力 (15%)**：中文理解、生成流畅度、常识覆盖
4. **逻辑推理 (15%)**：数学解题、多步骤思考、复杂推理能力
5. **长上下文 (10%)**：支持的上下文长度，长文档处理准确性
6. **代码能力 (10%)**：代码生成、调试、解释能力
7. **输出稳定性 (10%)**：幻觉率、事实准确性、输出一致性
8. **场景适配性 (5%)**：适合的使用场景丰富度

## 💡 Windows环境运行优化技巧
### 1. 显存利用最大化
```powershell
# 尽可能多的把模型层放到GPU显存
set OLLAMA_GPU_LAYERS=99
# 限制最大显存使用为15.5GB，避免显存溢出
set OLLAMA_GPU_MEMORY_FRACTION=0.95
```
### 2. 内存优化
```powershell
# 增加CPU内存缓存大小，提升长文档处理速度
set OLLAMA_CACHE_SIZE=16GB
```
### 3. 性能提升
- 关闭Windows虚拟内存页面文件，避免内存数据频繁换入换出
- 使用Ollama官方的Windows优化版本，比WSL2运行性能提升20%以上
- 模型文件存放在NVMe SSD上，加载速度提升3倍以上
