## ⚡ 终极解决技巧（万一还是差一点内存）
如果运行时还是提示内存不足，加上`--ctx-size 65536`参数把上下文窗口从128K降到64K，可以再省3-4GB内存，百分百能跑起来：
```powershell
# 示例：调低上下文运行qwen3:30b
ollama run qwen3:30b --ctx-size 65536
```
> 💡 64K上下文支持约5万字的文本处理，日常使用完全够用，不会影响普通场景的体验。
