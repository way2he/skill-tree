## 🛠️ 学习用快速命令

### 一次性读完所有 SKILL.md（PowerShell）
```powershell
$root = "C:\Users\robotAi\Documents\repo\skill-repo\superpowers\skills"
Get-ChildItem $root -Directory | ForEach-Object {
    $sf = Join-Path $_.FullName "SKILL.md"
    if (Test-Path $sf) {
        Write-Host "`n========== $($_.Name) ==========" -ForegroundColor Cyan
        Get-Content $sf -Raw -Encoding UTF8
    }
}
```

### 统计每个 skill 字数
```powershell
Get-ChildItem $root -Directory | ForEach-Object {
    $sf = Join-Path $_.FullName "SKILL.md"
    if (Test-Path $sf) {
        $wc = (Get-Content $sf -Raw).Split() .Count
        "{0,-40} {1}" -f $_.Name, $wc
    }
}
```

### 找包含某关键词的 skill
```powershell
Get-ChildItem $root -Recurse -Filter "SKILL.md" |
    Select-String -Pattern "你的关键词" |
    Select-Object Path, LineNumber, Line
```
